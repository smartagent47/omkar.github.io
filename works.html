Kernels

Megabyte Kernel

Megabyte was my first kernel project for Moto x based on @gustavo.s , @faux123 & @cyangogenmod kernel source with my spicy twist in it. The kernel was made for Lollipop 5.1 having the ability to save Battery & Powerful Performance

Hybrid Kernel

Hybrid was my 2nd project. It is not yet released. This kernel is based on @tycoo's caf 3.4.110 with Xperince kernel source. With his kernel, some features were added to it. Due to the inheritance, the kernel was named as Hybrid. The kernel is made for Android 6.0.x Marshmallow with advanced features & great Battery Performance
